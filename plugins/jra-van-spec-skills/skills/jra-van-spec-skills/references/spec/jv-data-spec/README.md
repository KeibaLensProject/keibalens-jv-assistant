## JVData の文字コード

- 全角文字: Shift_JIS (2バイト)
- 半角文字(英･数･半角ｶﾅ): JIS8 (1バイト)
- レコードの収録順序に仕様はなく、予告なく変わる場合がある。DB反映時はキー項目やデータ作成日を利用すること。
